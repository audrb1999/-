

넘파이의 기본 사용법

넘파이란? 다차원 배열을 효과적으로 처리할 수 있도록 도와주는 도구

현실 세계의 다양한 데이터는 배열의 형태로 표현 가능

파이썬의 기본 리스트에 비해 빠르고 강력한 기능을 제공

넘파이 때문에 데이터 처리에 용이 > 파이썬을 잘 사용하는 이유

 



여러가지 데이터가 나열될 때 이것을 매트릭스라 부름

1차운 > 행 >벡터

2차원 > 열 > 행렬

3차원 > 채널 > 텐서


파이참은 라이브러리 설치에 용이함!



일반 리스트의 경우

list_data = [1, 2]

array = np.array(list_data) > 이렇게 하면 알아서 넘파이 배열로 바뀜



0부터 3까지 배열 만들기

np.arange(4)

np.zeros((2, 2), dtype = float) .astype(float) > 이렇게도 가능



np.random.randint(0, 10, (3, 3))  > 0~9까지 랜덤하게 (3, 3) 배열로   배열을 만들때 최대값 -1 뺀값까지 라고 생각하기!


평균이 0이고 표준편차가 1인 표준 정규를 띄는 배열 만들기 > np.random.normal(0, 1, (3, 3))


넘파이 배열 두 개 합치기

컨케러네이트
np.concatenate([array, array], axis = 0) > 행쪽으로 합침 > 세로로


a, b = np.split(array, [2], axis = 1) > array를 열기준으로 2번 인덱스 기준으로 매트릭스를 나눈다는 의미


배열 합치기, 나누기도 되고 계산도 되고 형태로 reshape으로 바꿀 수 있음! 장점!
